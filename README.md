# HUST Simulator - Game Server

Hệ thống Backend Game Server Microservices dành cho HUST Simulator.

---

## 1. Hướng Dẫn Khởi Chạy

### Lệnh Khởi Chạy:
```bash
docker-compose up -d --build
```
